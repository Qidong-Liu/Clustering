# Clustering
This is our MATLAB implementation for the paper: Gobal Optimal Path-Based Clustering Algorithm (TPAMI under review)<br>
[URL](https://arxiv.org/abs/1909.07774)<br>
# Usage
```
Run clustering.m
```
# Data format

```
feature1 feature2 label
```
